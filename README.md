# Register App 

## Run project locally
### 1. Clone project
```bash
git clone https://github.com/nqq203/register-app.git
```

### 3.Create a .env file with these content
```bash
REACT_APP_API_BASE_URL=
-- Enter your url, normally REACT_APP_API_BASE_URL=localhost:8080
```

### 2. Install dependencies:
```bash
npm install
```

### 3. Start project
```bash
npm start
```
